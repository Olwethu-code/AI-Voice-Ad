# AI Voice AD

# This is a web application that uses Artificial Intelligence to convert promotional text or images into a professional advertisement with voice audio.

This platform is for bussinesses want to advertise their products, but they do not have skills or enough money to create professional advert.

This AI-powered tool is the solution that automatically generates a promo advertisement, the user can enter a text or upload an image.
The system then create a professional advert script and convert it into spoken audio.

# The system works in three ways;

First, the user enters a promo text or uploads an image

Second, the AI analyzes the information and generates a proffesional advertisement script

Third, the script is converted into voice audio that sounds like a real radio commercial.

# Technologies Used;

React and TypeScript for the Frontend

Node.js and Express for the Backend

PostgresSQL for the database

Artificial Intelligence models to generate scripts and audio.
